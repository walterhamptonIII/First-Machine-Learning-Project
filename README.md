# 🚢 Titanic: Survival Prediction & Model Iteration
### A Comparative Study of Random Forest and Support Vector Machines (SVM)

## 📌 Project Overview
This project is a deep-dive into the "Hello World" of Data Science: The Titanic Challenge. It documents my journey from basic data exploration to implementing complex classification algorithms, focusing on feature importance and model reliability.

## 🛠️ Tech Stack
* **Libraries:** Scikit-Learn, Pandas, Seaborn, Matplotlib
* **Models:** Random Forest (RF), Support Vector Machines (SVM)
* **Concepts:** Feature Engineering, Hyperparameter Tuning, Cross-Validation

## 🧬 Data Engineering Highlights
* **Imputation Strategy:** Analyzed age distributions to handle missing values without biasing the model.
* **Categorical Encoding:** Transformed qualitative features (Sex, Pclass) into model-ready numerical formats.
* **Algorithm Comparison:** Evaluated the performance gap between "ensemble-based" logic (Random Forest) and "distance-based" logic (SVM).

## 💡 The "Learning Journey" Insight
One of the key takeaways from this project was understanding why a model can achieve **85% accuracy on training data** but perform lower on **unseen Kaggle test data**. This led to a deeper investigation into **overfitting** and the importance of random states.
